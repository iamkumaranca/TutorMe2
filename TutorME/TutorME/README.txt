About the application

- Ability to ask questions within the TutorMe Community
- Get replies to TutorMe Community members
- Share the questions to Facebook and Twitter
- All members are registered members verified through Firebase

Technologies

- Firebase                              Backend solution
- Social Media Sharing                  Posting to Facebook and Twitter
- TableView Controllers                 Displaying info from database
- Tab Bar Controller
- Navigation Controller                 Used since the tab bar didn't look right when running app
- FontAwesome                           Customized icons
- MapKit                                About screen, displays map
